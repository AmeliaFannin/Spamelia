Spamelia, a Bayesian Spam Filter

When sent text, this API returns "Spam" or "Not-Spam".

Check out the code and a full README on github.

Test it out:

curl -X POST --header "Content-Type: application/json" http://spamelia.herokuapp.com/ -d '{"email_text": "Miracle diet cure, send money my dear friend"}'

Try my script for sending multiple emails and interpreting results, found in the utils folder on github.
